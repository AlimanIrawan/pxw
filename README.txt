Detik新闻爬虫独立版

这是一个独立的Detik新闻爬虫程序，专门用于从detik.com网站爬取印尼语新闻数据。

功能特点：
- 支持指定日期爬取新闻
- 智能时间解析（支持印尼语时间格式）
- 内容质量验证和清洗
- 支持多种输出格式（TXT、JSON、CSV）
- 图形化界面操作
- 详细的日志记录

使用方法：

1. 双击启动（推荐）：
   - 双击"启动爬虫.sh"文件
   - 系统会自动检查环境并启动GUI界面

2. 命令行启动：
   - 打开终端，进入程序目录
   - 运行：python3 main.py

3. GUI界面操作：
   - 选择目标日期（支持昨天、前天、今天快速按钮）
   - 选择输出格式（TXT、JSON、CSV）
   - 可选择输出目录
   - 点击"开始爬取"按钮
   - 在日志区域查看爬取进度

输出文件：
- 默认保存在output目录
- 文件名格式：detik_news_YYYY-MM-DD.txt
- 包含新闻标题、发布时间、内容、链接等信息

系统要求：
- Python 3.8+
- Chrome浏览器
- macOS系统

注意事项：
- 首次运行会自动创建虚拟环境并安装依赖
- 需要稳定的网络连接
- 爬取速度受网络和网站响应影响
- 建议不要频繁爬取，避免对目标网站造成压力

技术支持：
如有问题，请查看logs目录下的日志文件。
